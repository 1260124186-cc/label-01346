# 垃圾分类助手小程序 - 项目架构总结

## 一、项目概述

**垃圾分类助手**是一款帮助用户学习和掌握垃圾分类知识的微信小程序。项目采用纯前端架构，内置模拟数据，无需后端服务即可完整体验所有功能。

- **技术方案**：微信小程序原生框架（WXML + WXSS + JavaScript）
- **设计风格**：轻透清新风，环保主题
- **项目定位**：教育类工具小程序，兼具游戏化学习和积分激励机制

### 6.2 核心业务数据流

#### 兑换商品流程
```
用户点击商品 → 弹窗确认 → 检查积分/库存 → 二次确认
    ↓
扣减积分 (updateUserPoints) → 更新库存/销量
    ↓
持久化用户信息 (wx.setStorageSync) → 提示兑换成功
```

#### 答题积分流程
```
进入答题 → 选择答案 → 判断对错 → 错题入 wrongQuestions
    ↓
答完所有题 → 结算积分 → 增加积分 (updateUserPoints)
    ↓
保存章节进度 (chaptersProgress) → 展示结果页
```
