# 垃圾分类助手小程序

一款帮助用户学习和掌握垃圾分类知识的微信小程序。

## How to Run

- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)（最新稳定版）
- 微信小程序 AppID（可使用测试号）

### 启动步骤

1. **下载/克隆项目**
   ```bash
   git clone <repository-url>
   cd label-01346-main
   ```

2. **打开微信开发者工具**
   - 选择「小程序」项目
   - 点击「导入项目」

3. **配置项目**
   - 项目目录：选择 `frontend-mp` 文件夹
   - AppID：填入你的小程序 AppID（或点击「使用测试号」）
   - 项目名称：垃圾分类助手

4. **运行项目**
   - 点击「编译」按钮
   - 在模拟器中预览效果

5. **开发设置**（如遇图片加载问题）
   - 点击右上角「详情」
   - 勾选「不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书」

## 题目内容

一．产品设计目的 
现代社会，垃圾充斥在人们的生活，工作中。有些可以回收再利用，有些易腐，有些却是有害的。垃圾分很多种类，人们全部学习并掌握分类方法有困难，开发一款微信小程序可以对其进行辨别，去帮助我们更好地处理这些垃圾，可以在丢垃圾时归类投放，这样可以帮助相关工作人员减少工作量，也可以是我们的生活和环境变得更加美好。 
二．产品需求功能 
开发垃圾分类小程序部分功能：首页、积分兑换页面、个人资料页面、垃圾分类常识页面。其中首页主要用来展示垃圾分类小程序的各项功能，点击某种垃圾图片可以跳转到相应种类垃圾常识页面。积分兑换页面主要介绍获得的积分可以在商城进行兑换各种礼品。个人中心页面主要展示个人资料及一些基本信息的获取方式（其中头像功能是利用本地图片上传的效果）。

## 功能特性

### 1. 首页 (index)
- **轮播图展示**：使用 swiper 组件展示环保宣传图片，支持自动轮播
- **垃圾分类入口**：展示四种垃圾分类卡片（可回收垃圾、有害垃圾、厨余垃圾、其他垃圾）
- **快速导航**：点击"了解分类详情"跳转到分类常识页
- **点击跳转**：点击垃圾分类卡片跳转到对应分类详情页

### 2. 垃圾分类常识页 (classify)
- **参数接收**：接收首页传递的分类 ID 和名称参数
- **分类切换**：顶部可切换查看不同垃圾分类
- **分类说明**：展示当前分类的详细说明
- **常见垃圾**：展示常见垃圾示例列表，每种垃圾有对应的 emoji 图标
- **投放提示**：提供垃圾投放的注意事项
- **分享功能**：支持分享给微信好友

### 3. 积分兑换页 (exchange)
- **积分展示**：显示用户当前积分
- **积分规则**：点击查看积分获取方式和使用说明
- **活动轮播**：热门活动广告轮播
- **商品分类**：支持全部、热门、新品、限量分类筛选
- **商品列表**：一行两列商品展示，支持 scroll-view 滚动
- **兑换功能**：根据积分状态显示"立即兑换"、"积分不足"或"已售罄"
- **回到顶部**：滚动一定距离后显示回到顶部按钮

#### 积分规则

| 获取方式 | 积分 |
|---------|------|
| 每日签到 | +5 |
| 垃圾分类（每次） | +10 |
| 连续签到7天奖励 | +50 |
| 邀请好友注册 | +100 |
| 完成知识问答 | +20 |
| 分享小程序 | +10 |

### 4. 个人中心页 (profile)
- **用户信息**：展示头像、昵称、用户等级
- **头像上传**：支持从本地相册选择图片更换头像
- **数据统计**：展示分类次数、累计积分、连续打卡天数
- **等级系统**：根据积分显示用户等级和升级进度
- **功能菜单**：使用 WeUI 组件展示功能入口

### 5. 分类记录页 (records)
- **统计概览**：总分类次数、获得积分、连续天数
- **分类统计**：四种垃圾分类的次数统计
- **记录列表**：显示历史分类记录，包含垃圾名称、类型、积分、时间

### 6. 积分明细页 (points)
- **积分概览**：当前积分、累计获得、累计消费
- **筛选功能**：全部/获得/消费三种筛选
- **明细列表**：显示积分变动记录，包含来源、积分数、时间

### 7. 兑换订单页 (orders)
- **成功展示**：所有订单直接显示为兑换成功状态
- **订单列表**：显示商品信息、兑换积分、兑换时间
- **再次兑换**：支持快速跳转到积分兑换页

## 考核要点覆盖

| 考核要点 | 实现位置 | 说明 |
|----------|----------|------|
| 底部 tab 配置 | `app.json` | 配置了首页、积分兑换、我的三个 tab |
| window 导航配置 | `app.json` | 配置了导航栏颜色、标题、背景色 |
| swiper 轮播图 | `index.wxml`, `exchange.wxml` | 首页和积分兑换页都使用了 swiper 组件 |
| view/image 标签 | 所有页面 | 广泛使用了 view 和 image 基础组件 |
| bindtap 事件 | `index.js` | goToClassify 函数处理点击跳转 |
| wx:for 数据绑定 | `index.wxml` | 垃圾分类卡片列表使用 wx:for 循环渲染 |
| 页面参数传递 | `classify.js` | onLoad 中通过 options 接收页面参数 |
| scroll-view 组件 | `exchange.wxml` | 商品列表使用 scroll-view 实现滚动 |
| WeUI 框架 | `profile.wxml` | 使用 mp-cells、mp-cell 组件 |
| 头像上传 | `profile.js` | wx.chooseImage 实现本地图片上传 |

## 注意事项

1. **图片资源**：所有图片资源均使用本地文件，无外部依赖
   - 分类图标使用 Emoji，确保跨平台一致性
   - tabbar 图标使用本地 PNG 文件
   - 轮播图和商品图片使用本地 JPG 文件
2. **AppID 配置**：正式发布前请在 `project.config.json` 中替换为真实的 AppID
3. **WeUI 组件**：项目使用了微信官方 WeUI 扩展库，已在 `app.json` 中配置
4. **URL 编码**：页面跳转时中文参数会自动编码，接收时需要解码
